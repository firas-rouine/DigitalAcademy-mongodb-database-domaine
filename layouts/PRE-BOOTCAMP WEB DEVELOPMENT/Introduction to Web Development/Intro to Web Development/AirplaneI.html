    Airplane Game

    Welcome to the final level of our Introduction to Programming! Now it's time to take what we've learned, and create an Airplane which you'll be able to customize and play yourself!

    We will use all the same concepts we've learned so far:

    • Variables
    • Conditionals
    • Loops
    • Functions
    • Game Loop

    If you didn't get much practice on the game loop in Pacman, we will go over it together in this section, but in short, a game loop is simply a function we want to call every X amount of seconds. This can be used to move enemies, missiles, and also check for collisions

    Lets build together a basic Airplane game, and along the way, we will give you challenges to complete. After you've been through the entire game, we encourage you to build it following along, and later, try to customize what you have, or even build it on your own, most importantly, have fun with it!

    Part 1

    We will cover the following:

    • Build divisions for the ocean, players and enemies
    • Style the ocean.
    • Create player object to hold it's position.
    • Create enemies array, and objects for positions.
    • Create drawPlayer function to draw players.
    • Create drawEnemies function to draw enemies.
    • Create onkeydown trigger to move left and right.
