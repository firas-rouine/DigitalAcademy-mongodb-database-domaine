    Spot JS Patterns: Return

    Until now, we've used console.log() to display stuff to us as developers. However, that's not very useful for anyone else! Take a look below and see what patterns you might be able to see regarding this thing called "return".

    Code Snippet 1

    function getFirstNameFromForm (){
    var firstName = "Juan";
    return firstName;
}

    CHALLENGE! Code Snippet 2

    function calculateTipAmount (billTotal, tipPercent){
    var tipDue = billTotal * tipPercent;
    console.log("If you see this, this is just after tipDue was calculated");
    return tipDue;
}
calculateTipAmount(140, .20);

    1. Which of the following people would see a console.log() ?

    2. In Code Snippet 2, what does the returned value tipDue provide ?
