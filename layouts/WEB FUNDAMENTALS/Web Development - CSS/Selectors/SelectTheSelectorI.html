    Select the Selector I

    The following HTML code example may be needed by the questions of the quiz.

    <!DOCTYPE html>
<html >
<head>
    <meta charset="UTF-8">
    <title>Example</title>
</head>
<body>
    <div id="title">
        <h1>Hello World!</h1>
        <p>Subtitle to go along with the title</p>
    </div>
    <div class="main">
        <h3>New Topic</h3>
        <p class="quote">
            <a href="#" class="link">Lorem ipsum</a> dolor sit amet, consectetur adipisicing elit.<br>
            Ea dignissimos cum iure est ullam eum animi!
        </p>
    </div>
</body>
</html>

    1. Based on the code above, which selector would select the first div <div id="title">?

    2. Based on the code above, which selector would select the second div <div class="main">?

    3. Based on the code above, which selector would select the <h1> tag?

    4. Based on the code above, which selector would select all of the paragraph <p> tags?

    5. Based on the code above, which selector would select the anchor <a> tag?
