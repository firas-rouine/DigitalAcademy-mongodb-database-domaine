    Course Overview

    Learning Objectives:

    • Overview of the concepts covered in this course
    • Overview of the workload and cadence of this course

    Welcome to the Java and Spring Course. This course will take you from the basics of the Java programming language to building full-scale web applications ready for deployment. The course is separated into several units:

    • Java Fundamentals
    • Object Oriented Programming with Java
    • Java Spring
    • Java Spring Deployment

    Throughout the course, you will learn about the Java programming language and Spring, a framework for developing Java applications. Through these, you will get a deeper understanding of programming and web development, making you a more rounded full-stack developer.

    Course Overview

    Java is a challenging but very rewarding language to learn. Here's a preview of the concepts you'll be learning in this course. The assignments are designed to build on one another. For new concepts, you will first practice each with smaller assignments and then combine multiple competencies in a larger core assignment. Working through a core assignment is designed to synthesize previous skills to gain proficiency on a topic in order to move on.

    Workload

    Before you get started here is a quick note on how this curriculum has been laid out as well as a subjective gauge of the work load each day relative to the Belt Exam.

    Cadence

    As you can see, we will be spending the first few days getting acquainted with the Java programming language and delving deeper into Object Oriented Programming. Next, we start learning how to make Web applications using Spring Boot. Initially we will start without a full MVC structure. We will then introduce Spring Data, so that we can work with databases using JPA. Next, we will be learning how to make One-to-Many relationships, then will add users and Login and Registration. Finally, we will study Many-to-Many relationships. At this point you will be ready to attempt the belt exam. The belt exam is undeniably a challenging milestone, but if you have used your time well, you can and will be successful.

    In preparation for developing a more sophisticated project during project week, we will also cover how to make more advanced queries and spend some time talking about Spring Security.

    Disclaimer:

    This information is an Overview but remains flexible. The above is subject to change, and sometimes rearrangement depending on the needs of the class. Please check with your instructor if you want a more exact timeline. It is worth noting a common pitfall is for students to take lower workload day as a break from study. This almost always leads to a loss of momentum and for some, getting irrecoverably behind. Instead, use the time to stay on pace or do optional assignments if you are ahead. Every day is critical for your understanding of Java, for preparation for the Belt Exam, and more importantly, for building fully functional full-stack applications.

    Why Learn Java?

    Java is one of the most widely used programming languages and is estimated to have around 10,000,000 developers worldwide. Given that there are around 24,000,000 software developers worldwide - it is a sizeable chunk.

    Aside from its popularity, the fact that it can run on almost every computer without changing behavior, as well as its huge selection of libraries and tools, makes it the first choice of languages for many developers. It has also long been a mainstay of educational institutions due to the concepts it teaches and its interoperability between systems.
