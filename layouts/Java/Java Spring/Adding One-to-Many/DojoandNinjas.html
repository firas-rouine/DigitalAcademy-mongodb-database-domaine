    Dojo and Ninjas Assignment

    Learning Objectives:

    • Implement a one-to-many relationship
    • Manipulate lists to model one-to-many relationships in the database
    • Implement a foreign-key relationship by assigning instances of a class as a member variable

    In this assignment, you will be creating an application that will keep track of dojos and all the ninjas that are part of a specific dojo.

    ERD and Wireframe



    Allow users to create a new dojo.

    Allow new ninjas to choose their dojo via a dropdown.

    Create a dojo page that displays all ninjas that belong to that specific location.
