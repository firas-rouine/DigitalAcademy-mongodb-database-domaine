Table Master Exam

In this example, you will create a table master where waitstaff can log in to add, edit, and delete tables.

For a Red Belt:

  • The waitstaff can create an account and become a registered user.
  • Dashboard: Create a dashboard where waitstaff can view all tables.
  • Waitstaff can add a new table.
  • Waitstaff can edit their information.
  • Waitstaff can delete their tables.
  • Add validations for:
    • Registration process
    • Login process
    • Table creation process

For a Black Belt:

  • Include a 'Give Up Table' link that allows the user to give up the table for another user to pick up. The table should disappear from this grid but appear on the /tables page.

  • Include a link to assign the logged-in user as the server for this table. The table should disappear from this grid (and appear on the home page).
