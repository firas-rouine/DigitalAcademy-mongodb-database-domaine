Course Platform - Instructors Exam

In this example, you will create a Course Platform where instructors can log in to add, edit, and delete courses.

For a Red Belt:

  • The instructor can create an account and become a registered user.
  • Dashboard: Create a dashboard where instructors can view all courses.
  • Instructors can add their courses to the schedule.
  • Instructors can edit their course information.
  • Instructors can delete their courses.
  • Add validations for the:
    • Registration process
    • Login process
    • Course creation process

For a Black Belt:

  • Provide the time of the courses in the dashboard.
  • Include a time field in the create and edit course form.
  • Allow adding new and existing students to each course, and display a list of students in the course information.
