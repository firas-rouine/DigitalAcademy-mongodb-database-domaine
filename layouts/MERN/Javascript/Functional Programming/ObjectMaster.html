    For example we could create a list of pokmon that have names that start with the letter "B" by using the following code.


    const bListPkmn = pokmon.filter( p => p.name[0] === "B" );


    Or if we wanted to return an array of just the ids, we could use something like this. 


    const pkmnIds = pokmon.map( p => p.id )



    Using the above pokmon array, find the following:


    an array of pokmon objects where the id is evenly divisible by 3.

    an array of pokmon objects that are "fire" type.

    an array of pokmon objects that have more than one type.

    an array with just the names of the pokmon.

    an array with just the names of pokmon with an id greater than 99.

    an array with just the names of the pokmon whose only type is poison.

    an array containing just the first type of all the pokmon whose second type is "flying".

    a count of the number of pokmon that are "normal" type.
